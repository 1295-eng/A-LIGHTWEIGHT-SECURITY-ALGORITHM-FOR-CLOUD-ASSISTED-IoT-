A node-level power-aware security mechanism is used in the proposed system. As a result, a secure IoT using our technology might withstand most data confidentiality threats from both insiders and outsiders of the IoT. Lightweight security methods for low-power IoT devices with smaller key sizes, lower power consumption, and lower memory and transmission capacity reserve are used to achieve security.

A node-level implementation of the public key cryptosystem, Elliptic Curve Cryptography, which was considered as the efficient algorithm based on research, is being implemented to achieve efficient security for low power IoT devices.
