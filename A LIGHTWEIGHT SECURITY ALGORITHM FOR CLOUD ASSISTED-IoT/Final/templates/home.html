A LIGHTWEIGHT SECURITY ALGORITHM FOR CLOUD ASSISTED-IoT


The Internet of Things, which is the fastest technology, will boost medical electronics, agriculture, and the defense sector. IoT is all on discovering a purpose for every object. The cloud assisted IoT provides a possible solution for data growth problems. However, as result of clouds influence, IoT now faces new security concerns for data mutuality between two parties. This work conducts a systematic investigation using a secure cloud assisted IoT data management approach to maintain data confidentiality when collecting, storing and accessing IoT data using the cloud while taking into account the constant rising users. So, an optimized node level power aware security mechanism is used in the proposed system. As a result, a secure IoT using our technology might withstand most data confidentiality threats from both insiders and outsiders of the IoT. Using a heartbeat sensor and a node MCU, create a heartbeat monitoring system. At the node MCU level, giving security to the patient's health data and preventing unauthorized users from attacking it. Lightweight security methods for low-power IoT devices with smaller key sizes, lower power consumption, and lower memory and transmission capacity reserve are used to achieve security. In order to achieve this, RSA algorithm results have been tabulated for this experimental setup to measure the performance.
